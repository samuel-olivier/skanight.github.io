FDF

Description

The FDF is a representation of a 3D grid where the 3D is handled using a simple perspective projection from 3D to 2D in order to draw it on a HTML5 2D Canvas.

How to use?

  • Move : , , ,
  • Zoom : Ctrl + /
  • Select a vertex : Mouse
  • Move a vertex : a, d, w, s, q, e
