Clother

I developed this Cloth Simulation software and its basic real-time renderer for the CSE 169 class at UCSD (University of California, San Diego) during the Winter 2014 quarter. I simulate the cloth using the spring damper technique and I compute the gravity, aerodynamic forces (wind), collisions with a ball and the floor.
During this project, I also learned to implement a whole parallel computing technique using OpenCL.
